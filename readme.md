# Tesla Inspired By her

## Stack 

- Vue.js
- Babel

## Requirements

- [ ] NodeJS

## Install

```bash

git clone https://github.com/baloran/tesla-her.git

cd tesla-her

npm i
```

## Start the awesome

```bash

npm run dev
```
